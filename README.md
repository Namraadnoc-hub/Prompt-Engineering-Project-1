# Smart Resume Information Extractor using Prompt Engineering

## Project Overview

This project was completed as part of the DecodeLabs Industrial Training Program 2026.

The objective of this project is to design a deterministic prompt that converts unstructured resume text into structured JSON using Prompt Engineering techniques.

## Features

- Zero-Shot Prompting
- Few-Shot Prompting
- Strict JSON Output
- Delimiter-Based Prompting
- Resume Information Extraction
- Missing Value Handling

## Technologies Used

- Prompt Engineering
- Large Language Models (LLMs)
- JSON
- Visual Studio Code
- GitHub

## Project Files

- `prompt.txt` – Main prompt with instructions and examples
- `sample_input.txt` – Sample unstructured resume
- `output.json` – Generated structured JSON output
- `README.md` – Project documentation

## Sample Workflow

1. Open the prompt.
2. Paste the unstructured resume.
3. Run the prompt using an LLM.
4. Receive structured JSON as output.
